+   Launch Appmethod.
+   Open dpk file. At `../mock1/base/CmMock.dpk`.

+   Right click **CmMock.bpl** and choice Install!
+   Open Tool > Option > Object Pascal Option > Library.
+   Add libirary path `..mock1/base`.
